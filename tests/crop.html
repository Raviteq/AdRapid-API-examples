Image crop example

Sample implementation of the image crop functionality.

Documentation is available http://raviteq.github.io/adrapid-api/#upload

This perticular example is using the imgAreaSelect library for the crop tool frontend.

These values will be posted to the crop method. In a real-world scenario, the form field will be hidden for the end user. We keep track of the width of the thumbnail to be able to send crop values as percentages.

These values are for illustrative purposes only
