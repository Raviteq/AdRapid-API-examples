HTML5 live preview

This example illustrated how dynamic previews for html5 banners can be acheived.

This consists of two steps. First we need to get a preview of the banner for a specific template and embed the preview into the active page.

Secondly, we generate a form from the rules of the given template. We then bind form change events to update the preview when the form fields (texts, images, colors) are changed. This example is also using upload helpers and color pickers.

The getForm helper function handles getting the rules for a templateId, as well as generating a form using the template rules. The getLivePreview helper function handles binding of form events, as well of getting the preview. Check the source code of the helpers.js library for further details on how this is achieved.
